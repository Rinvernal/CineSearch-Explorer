# 🎬 CineSearch — Premium Movie Explorer

**CineSearch** is a sophisticated React-based web application designed for cinema enthusiasts. This project evolved from a standard academic task into a high-end digital product, showcasing advanced UI/UX principles and modern frontend development techniques.

🚀 **[Live Demo](https://cinesearch-rinvernal.vercel.app/)**

---

## 💎 Design Highlights (UI/UX)
The application is crafted using a **Glassmorphism** aesthetic, providing a premium, cinematic feel:

* **Glassmorphic Header:** Semi-transparent navigation with a high-end `backdrop-filter: blur` effect.
* **Contextual Backgrounds:** The Movie Details page dynamically adapts its background based on the specific film's backdrop for an immersive experience.
* **Interactive Components:** Movie cards feature smooth hover animations, tailored rating badges, and lime-accented highlights.
* **Sleek Navigation:** Professional tabbed interface for seamless switching between Cast and Reviews.

---

## 🛠️ Technical Stack
* **React 18** — For building a dynamic and responsive user interface.
* **React Router DOM** — Advanced routing for a Multi-Page Application (MPA) feel (Home, Search, Details, Sub-routes).
* **TMDB API** — Integration with a global movie database for real-time data fetching.
* **CSS Modules** — For modular, scoped, and maintainable styling.
* **Formik** — Robust form handling for the search functionality.
* **Axios** — Efficient and reliable API communication.

---

## 📈 Evolution & Refactoring
This repository serves as a prime example of successful project transformation. Key improvements include:
1.  **Layout Modernization:** Migrated from basic lists to complex, responsive layouts using **CSS Grid and Flexbox**.
2.  **Design System Implementation:** Established a consistent visual identity (Montserrat typography and a signature lime-accented color palette).
3.  **Code Architecture Optimization:** Refactored components for better reusability and scalability.

## ⚙️ Installation & Setup
1. Clone the repository: `git clone https://github.com/Rinvernal/CineSearch-Explorer.git`
2. Install dependencies: `npm install`
3. Launch the development server: `npm run dev`
